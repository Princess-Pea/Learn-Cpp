# STL (Standard Template Library, 标准模板库)

### cplusplus.com

一个在线的C++参考网站，提供了丰富的C++标准库函数和类的文档，包括函数的用法、参数说明、返回值等信息。它是一个非常有用的资源，可以帮助开发者快速查找和理解C++标准库中的各种功能和特性。

### CpppReference.com

另一个在线的C++参考网站，提供了详细的C++标准库函数和类的文档，包括函数的用法、参数说明、返回值等信息。它还提供了示例代码和用户评论，可以帮助开发者更好地理解和使用C++标准库中的各种功能和特性。CpppReference.com也是一个非常有用的资源，尤其对于需要深入了解C++标准库细节的开发者来说。

### gcc.gnu.org

GNU Compiler Collection（GCC）的官方网站，提供了GCC编译器的下载、文档和支持资源。GCC是一个开源的编译器套件，支持多种编程语言，包括C、C++、Fortran等。gcc.gnu.org提供了GCC的最新版本、安装指南、用户手册和开发者资源，是使用GCC编译器的开发者的重要参考网站。

## STL六大部件 (Components)

本质都是模板类或函数。
容器(Containers)：用于存储和管理数据的容器类
分配器(Allocator)：负责内存分配和释放的类
算法(Algorithms)：提供各种数据操作算法的函数模板
迭代器(Iterator)：用于遍历容器中元素的对象
适配器(Adapters)：对现有容器或算法进行包装，以提供不同的接口
仿函数(Functors)：重载了函数调用操作符的类，可以像函数一样调用
